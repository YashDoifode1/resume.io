================================================================================
                          TROUBLESHOOTING
================================================================================

ISSUE: "PDF library not installed"
SOLUTION:
   - Run installation script
   - Or: composer require dompdf/dompdf
   - Verify vendor/autoload.php exists

ISSUE: "Class not found: Dompdf\Dompdf"
SOLUTION:
   - Run: composer install
   - Check vendor/autoload.php exists
   - Restart web server

ISSUE: "PDF downloads but is blank"
SOLUTION:
   - Increase memory_limit to 256M
   - Increase max_execution_time to 60
   - Restart web server

ISSUE: "Images not showing in PDF"
SOLUTION:
   - Enable GD extension
   - Check image file permissions
   - Try uploading image again

ISSUE: "PDF takes too long"
SOLUTION:
   - Increase memory_limit to 256M
   - Increase max_execution_time to 60
   - Use simpler theme
   - Optimize images
