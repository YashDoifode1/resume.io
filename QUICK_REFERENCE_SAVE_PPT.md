## API Endpoints

### Save Section
- **URL**: `/api/save-section.php`
- **Method**: POST
- **Data**: JSON with section name and data

### Download PPT
- **URL**: `/api/download-ppt.php`
- **Method**: GET
- **Response**: PowerPoint file
