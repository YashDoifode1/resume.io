================================================================================
                          INSTALLATION QUICK START
================================================================================

STEP 1: Copy Files
- Copy resume folder to C:\xampp\htdocs\ (Windows)
- Or /var/www/html/ (Linux)

STEP 2: Set Permissions
- chmod 755 resume/ (directories)
- chmod 644 resume/*.php (files)

STEP 3: Update Configuration
- Edit config/constants.php
- Update BASE_URL
- Update contact information
- Update social links

STEP 4: Access Application
- Open browser
- Go to http://localhost/resume/
- Click "Resume Builder"
- Start creating resumes!

================================================================================
                          USAGE INSTRUCTIONS
================================================================================

TO CREATE A RESUME:
1. Click "Resume Builder" in navigation
2. Fill in personal information
3. Add work experience, education, skills, etc.
4. Upload profile picture (optional)
5. Click "Preview Resume"
6. Select different themes
7. Click "Download PDF"

TO CUSTOMIZE:
1. Edit config/constants.php for configuration
2. Edit assets/css/variables.css for colors
3. Edit theme files in themes/ for design
4. Edit pages/ for website content

TO DEPLOY:
